# LXShare


## 介绍：

1:LXToolManger常规的工具


2:NSDecimalNumber 字符串的加减乘除（NSDecimalNumber高精度的数据处理)


3:NSMutableAttributedString 字符串前添加图片


4:UIAlertController 弹窗的封装处理


5:常用UI的扩展



## 安装



```ruby
 pod 'LXShare', :git => 'https://github.com/hfdz1011/LXShare.git'
```

## 使用


```ruby
 #import "LXMangerHeader.h"
```
